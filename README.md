# 🌦️ Weather Data Analysis & Temperature Prediction

This project analyzes historical weather data and predicts future temperature trends using Machine Learning.

## Features
- Synthetic weather dataset generation
- Data preprocessing
- Linear Regression model
- Temperature prediction
- Visualization of results

## Tech Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## How to Run

```bash
pip install -r requirements.txt
python download_weather.py
python weather_predictor.py
